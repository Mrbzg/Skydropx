============================================================
   AGENTE FENIX v5  —  COMO INSTALAR (super facil)
============================================================

No necesitas saber nada tecnico.


============================================================
  WINDOWS  (lo mas facil)
============================================================

  1. Doble clic en:   INSTALAR-WINDOWS.bat

  2. Aparece un aviso de permisos (UAC). Haz clic en "SI".
     (El instalador necesita permisos para configurarse.)

  3. Espera a que diga "TODO LISTO" (1-3 minutos la primera vez).

  4. Abre opencode (o Claude Code) y escribe:   /skills
     Veras "agente-fenix". Listo.


  -- SI WINDOWS SIGUE BLOQUEANDO EL ARCHIVO --
  (Mensaje: "Control inteligente de aplicaciones bloqueo...")

  Windows desconfia de TODO archivo descargado de internet, aunque
  sea seguro. La forma de evitarlo es desbloquear el ZIP ANTES de
  extraerlo (asi salen desbloqueados todos los archivos de golpe):

     1. Clic DERECHO en el archivo ZIP que descargaste
     2. "Propiedades"
     3. Abajo marca la casilla "Desbloquear" (Unblock)
     4. "Aceptar"
     5. AHORA extrae el ZIP y haz doble clic en INSTALAR-WINDOWS.bat

  Si ya lo extrajiste, tambien puedes hacer lo mismo (clic derecho ->
  Propiedades -> Desbloquear) directamente sobre INSTALAR-WINDOWS.bat.

  O si aparece la ventana azul de SmartScreen:
     1. Clic en "Mas informacion"
     2. Clic en "Ejecutar de todas formas"


============================================================
  MAC / LINUX
============================================================

  1. Doble clic en:   INSTALAR-MAC-LINUX.command
     (En Mac, si dice que no se puede abrir: clic derecho -> Abrir.)
  2. Espera a que diga "TODO LISTO".
  3. Abre opencode (o Claude Code) y escribe:   /skills


============================================================
  OPCION ALTERNATIVA (decirle al asistente)
============================================================

  Abre opencode o Claude Code DENTRO de esta carpeta y escribe:

       instala el agente fenix

  El asistente hara todo solo. (Esta forma nunca la bloquea Windows.)


============================================================
  ¿NECESITAS MAS POTENCIA? (instalador PRO, opcional)
============================================================

  Con el instalador normal manejas hasta ~10,000 leads por sesion.

  Si vas a hacer corridas MAS PESADAS (mega-corridas, OSINT
  profundo, analytics rapidos), instala las capacidades PRO:

    WINDOWS:    doble clic en  INSTALAR-PRO-WINDOWS.bat
    MAC/LINUX:  python install-pro.py

  El instalador PRO se auto-evalua (ve que tienes y que falta),
  se auto-repara, e instala solo lo que haga falta. Si algun
  paquete pesado no se puede instalar, lo omite SIN romper nada.

  No es obligatorio. Solo instalalo si lo necesitas.


============================================================
  IMPORTANTE
============================================================
  - Solo se instala UNA vez. Despues, el agente fenix
    aparece en /skills SIEMPRE, en cualquier carpeta.
  - Si opencode ya estaba abierto cuando instalaste,
    cierralo y abrelo de nuevo.
  - Necesitas Python instalado (https://www.python.org/downloads/
    -> marca "Add Python to PATH"). Si ya lo tienes, mejor: es mas rapido.
  - Para usarlo, solo pide cosas como:
       "corre fenix para 50 leads de ropa en CDMX"
       "verifica el telefono 5512345678"
